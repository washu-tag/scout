# Scout Diagrams

This directory contains source files for generating architecture diagrams used in the documentation.

## Files

- `scout-architecture.html` - Interactive HTML diagram of the Scout platform architecture
- `logos/` - Logo assets used by the HTML diagram

## Regenerating the Screenshot

1. Open `scout-architecture.html` in a browser
2. Take a screenshot of the diagram
3. Save as `docs/source/images/ScoutArchitecture.png`

The screenshot is referenced in `docs/source/services.md`.
